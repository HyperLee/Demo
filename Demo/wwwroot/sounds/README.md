# 燈箱音效檔案說明

## 需要的音效檔案

### 🔊 開啟音效 (modal-open)
- **檔案名稱**: `modal-open.mp3`, `modal-open.ogg`
- **音效類型**: 溫和的「叮」聲或上升音調
- **時長**: 0.3-0.5 秒
- **音量**: 適中
- **建議**: 輕快、正面的提示音

### 🔇 關閉音效 (modal-close)
- **檔案名稱**: `modal-close.mp3`, `modal-close.ogg`  
- **音效類型**: 輕柔的「咔」聲或下降音調
- **時長**: 0.3-0.5 秒
- **音量**: 適中
- **建議**: 溫和、完成感的提示音

## 📁 檔案需求
- 格式：MP3 和 OGG (跨瀏覽器相容性)
- 大小：每個檔案 < 50KB
- 品質：適中，避免過大檔案影響載入速度

## 🎵 取得音效資源
1. **免費資源**: Freesound.org, Zapsplat.com
2. **線上產生器**: SFXR, Bfxr
3. **購買資源**: AudioJungle, Pond5

請將音效檔案放置在此目錄下，檔名需符合上述命名規範。
